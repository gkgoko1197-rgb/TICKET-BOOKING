# Project Tech Stack

This document outlines the primary languages, frameworks, and libraries used in this project.

## Front-end

-   **Language**: TypeScript, JSX
-   **Framework**: React (with Next.js)
-   **UI Components**: ShadCN UI
-   **Styling**: Tailwind CSS
-   **Icons**: Lucide React
-   **Forms**: React Hook Form with Zod for validation
-   **Carousel**: Embla Carousel

## Back-end (Server-side)

-   **Language**: TypeScript
-   **Framework**: Next.js (App Router, Server Components, Server Actions)
-   **Runtime**: Node.js
-   **AI Integration**: Google Genkit

## Tooling & Build

-   **Package Manager**: npm
-   **Build/Development**: Next.js CLI (`next dev`, `next build`)
-   **Type Checking**: TypeScript

## Other Key Libraries

-   `zod`: For schema validation on both client and server.
-   `date-fns`: For date formatting and utilities.
